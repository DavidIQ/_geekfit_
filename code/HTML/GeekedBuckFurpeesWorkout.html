Geeked Buck Furpees Workout

  • 12 burpees

  • 10 burpees
  • 400 meter run
  • 9 burpees
  • 400 meter run

This workout might be slightly long depending on your running stamina. The running can be done on a treadmill, but be prepared to occupy one for 30-45 minutes. After each run perform burpees for 1 less rep than last time. Continue reducing until you reach 1 burpee. By the end of this workout you would have run for 3 miles and done 78 burpees.
